# Typescript Oldschool
Boilerplate built in the spirit of progressive enhancement and performance.

## Tech Stack

- Node.js
- TypeScript
- Pug for HTML templating
- SCSS for CSS preprocessing
- Workbox for PWA management
- Mocha for testing
- Cypress for E2E
